# 🎨 Museum of Candy

A responsive, multi-section landing page for a fictional "Museum of Candy" built using HTML, CSS, Bootstrap 4, and jQuery. This project is part of my journey in front-end development and web design.

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Custom styling
- **Bootstrap 4** – Responsive layout
- **jQuery** – Basic DOM interactivity (scroll effect)

---

## 📐 Features

- Fully responsive layout using Bootstrap grid
- Mobile-friendly navigation bar with toggler
- Scroll-triggered navbar color change
- Visual-themed candy sections with images and colored branding
- Modular and semantic HTML + organized custom CSS
